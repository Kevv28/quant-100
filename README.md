# 100 Days 100 Quant Strategies

This repository documents my journey of building 100 systematic,
rule-based quantitative trading strategies in 100 days.

## Objective
- Learn market behavior through data
- Convert ideas into testable strategies
- Build discipline in quantitative research

## Structure
- Each day = one independent strategy
- Each strategy includes:
  - Hypothesis
  - Rules
  - Backtest
  - Results
  - Failure analysis

## Tech Stack
- Python
- pandas, numpy
- matplotlib

## Disclaimer
For learning and research purposes only.
